# 🚗 Car Price Prediction — Machine Learning Project  
### Prevendo preços de carros com modelos de regressão

Este projeto utiliza técnicas de **Machine Learning supervisionado** para prever o preço de veículos com base em características como quilometragem, ano, marca, modelo, potência, consumo e outros atributos relevantes.

O objetivo é construir um modelo capaz de estimar valores de mercado com boa precisão, auxiliando concessionárias, plataformas de venda e consumidores.

---

## 🚀 Objetivos do Projeto

- Explorar o dataset e entender os fatores que influenciam o preço de veículos  
- Realizar limpeza, tratamento e engenharia de atributos  
- Testar múltiplos modelos de regressão  
- Comparar métricas e selecionar o melhor modelo  
- Criar visualizações para interpretar padrões e relações  
- Documentar o processo de forma clara e profissional  

---

## 🛠️ Tecnologias Utilizadas

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 🔍 Etapas do Projeto

### **1. Análise Exploratória (EDA)**
- Distribuição dos preços  
- Relação entre preço e quilometragem  
- Impacto do ano de fabricação  
- Comparação entre marcas e modelos  
- Identificação de outliers  

### **2. Pré-processamento**
- Tratamento de valores ausentes  
- Codificação de variáveis categóricas  
- Normalização/Padronização  
- Feature engineering (ex.: idade do veículo)  

### **3. Modelagem**
Modelos testados:

- Regressão Linear  
- Random Forest Regressor  
- Gradient Boosting Regressor  
- XGBoost (opcional)  
- KNN Regressor  

### **4. Avaliação**
- RMSE  
- MAE  
- R²  
- Comparação entre modelos  
- Análise de erros
